# Sales Data Analysis (Python)

## Overview

This project analyzes a sample sales dataset to identify trends in customer spending, sales performance, and behavior.

## Tools Used

* Python
* Pandas
* Matplotlib

## Key Questions

* Which days generate the most sales?
* Does time of day affect revenue?
* How does group size impact spending?
* How do tipping patterns vary?

## Key Insights

* Sales are highest on weekends, especially Saturday.
* Dinner generates more revenue than lunch.
* Larger groups contribute more to total sales.
* Tip percentages are relatively consistent across days.
